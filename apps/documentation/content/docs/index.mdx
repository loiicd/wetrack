## What is WeTrack?

WeTrack is a **Dashboard-as-Code** platform. You define dashboards, data sources, queries, and charts as TypeScript — then deploy with one CLI command.

**Your dashboards live in your repo.** Version controlled, code reviewed, CI/CD deployed.

## Prerequisites

- Node.js ≥ 18 or [Bun](https://bun.sh) ≥ 1.2
- A WeTrack account → [Sign up](https://app.wetrack.dev/signUp)
- An API Key → Settings → API Keys

## Quickstart

### 1. Install the SDK

```bash
npm install wetrack-dashboard
```

### 2. Create your first stack

Create `mystack.ts`:

```typescript
import { Stack, Dashboard, DataSource, Query, Chart } from "wetrack-dashboard";

export default new Stack("my-first-stack", "PRODUCTION")
  .addDashboard(
    new Dashboard("main", { label: "My Dashboard" })
  )
  .addDataSource(
    new DataSource("api", {
      type: "rest",
      config: {
        url: "https://jsonplaceholder.typicode.com/todos",
        method: "get",
      },
    })
  )
  .addQuery(
    new Query("todos", {
      type: "jsonpath",
      dataSource: "api",
      jsonPath: "$[*]",
    })
  )
  .addChart(
    new Chart("status-chart", {
      dashboard: "main",
      source: { _entity: "query", key: "todos" },
      label: "Todos by User",
      type: "bar",
      config: {
        categoryField: "userId",
        valueFields: ["id"],
        orientation: "vertical",
      },
      layout: { x: 0, y: 0, w: 12, h: 3 },
    })
  );
```

### 3. Deploy

```bash
export WETRACK_API_KEY=<your-api-key>
npx wetrack-cli deploy mystack.ts
```

Your dashboard is now live at [app.wetrack.dev](https://app.wetrack.dev) 🎉
