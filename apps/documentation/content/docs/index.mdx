## What is WeTrack?

WeTrack is a **Dashboard-as-Code** platform. You define dashboards, data sources, queries and charts as TypeScript code, then deploy them with a single CLI command.

## Prerequisites

- [Bun](https://bun.sh) ≥ 1.2 or Node.js ≥ 18
- A WeTrack account → [Sign up](https://app.wetrack.io/signUp)
- An API Key (Settings → API Keys)

## Installation

```bash
bun add -g @wetrack/cli
```

## Your first stack

Create `mystack.ts`:

```typescript
import { Stack, Dashboard, DataSource, Query, Chart } from "dashboard_as_code";

const stack = new Stack("my-first-stack", "PRODUCTION")
  .addDashboard(new Dashboard("main", { label: "My Dashboard" }))
  .addDataSource(new DataSource("api", {
    type: "rest",
    config: { url: "https://jsonplaceholder.typicode.com/todos", method: "get" }
  }))
  .addQuery(new Query("todos", {
    type: "jsonpath",
    dataSource: "api",
    jsonPath: "$[*]"
  }))
  .addChart(new Chart("status-chart", {
    dashboard: "main",
    query: "todos",
    label: "Todos by User",
    type: "bar",
    config: { categoryField: "userId", valueFields: ["id"] }
  }));

export default stack;
```

## Deploy

```bash
export WETRACK_API_KEY=<your-api-key>
wetrack deploy mystack.ts \
  --url https://app.wetrack.io/api/dashboard
```

Open your dashboard at `https://app.wetrack.io` 🎉
